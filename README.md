# HFTuner
The hands-free guitar tuner!

Utilizes the TarsosDSP API
https://github.com/JorenSix/TarsosDSP  

## Compiling

### Windows 
`javac .;bin/TarsosDSP-latest.jar *.java`  

### Linux
`javac .:bin/TarsosDSP-latest.jar *.java`  

## Running

### Windows 
`java .;bin/TarsosDSP-latest.jar HFTuner`

### Linux
`javac .:bin/TarsosDSP-latest.jar HFTuner`
